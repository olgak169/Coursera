1. Download and install Visual Studio Code and GitHub Desktop 2. At some point GHD will ask you to authorize it to your github account 3. On github webpage in your account find repositories tab, click New. 4. In the page that opens under Quick setup option there should be and icon Set up Desktop. Click that, it should open GHD app on your computer. 5. Window with file paths will open. Either change path or leave as is. 6. On the right side of the app Open the repository in your external editor will be there. Set up VSCode in Preferences, and click Open in VSCode. 7.VSCode might ask you to install additional packages to connect to github, but that should not be an issue, it will install them automatically if you press yes. 8. Once in VSCode, you can create your index.html and whatnot right there. 9. When changes are made to the files and you want to send files to your github, bottom left corner is a small icon with arrows, press that. 10. In GHD press commit to master
